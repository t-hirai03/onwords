本番サイト: https://www.onwords.co.jp/company

## ページ構成

会社概要ページは以下の5つのセクションで構成されています：

1. **hero** - ヒーローセクション
2. **board-member** - 経営陣紹介
3. **mission** - ミッション
4. **company-info** - 会社概要詳細
5. **access** - アクセス（Google Maps）

## 共通スタイリング

### セクションヘッダー

すべてのセクションで共通のヘッダースタイル：

- ラベル: 16px / 15px（モバイル）、赤色（rgb(230, 1, 18)）
- タイトル: 36px / 32px（モバイル）、黒色（rgb(34, 34, 34)）
- 中央揃え（text-align: center）
- 縦並び（flex-direction: column）

### カラーパレット

- **赤（アクセントカラー）**: rgb(230, 1, 18)
- **黒（見出し）**: rgb(34, 34, 34)
- **グレー（テキスト）**: rgb(51, 51, 51)
- **ライトグレー（ボーダー）**: rgb(238, 238, 238)
- **白**: rgb(255, 255, 255)

## WordPress実装時の注意点

1. **プロフィール画像**: 各経営陣の画像は `assets/images/company/` に配置
2. **Google Maps**: iframeの埋め込みコードを使用
3. **企業理念リンク**: `/company/philosophy` ページを作成
4. **レスポンシブ**: 全ブレークポイントでテスト必須
5. **BEM命名規則**: クラス名は `.company-{section}__{element}` 形式
