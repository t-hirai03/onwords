# Hero Section - HTML

```html
<div class="company-hero">
  <p class="company-hero__label">Company</p>
  <h1 class="company-hero__title">会社概要</h1>
</div>
```

## 構造

- 外側: `div` - Flexコンテナ（縦方向・中央揃え）
- 内側:
  - `p` - 英語ラベル "Company"
  - `h1` - 日本語タイトル "会社概要"
